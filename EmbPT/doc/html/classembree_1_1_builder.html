Detailed Description

Builder base class.

This class implements an efficient multi-threaded memory allocation scheme used by the builders. A builder thread can request new memory for a number of nodes or triangles (threadAllocXXX function). The per thread allocator allocates from its current memory block or requests a new block from the slower global allocator (globalAllocXXX function) when this block is full.
