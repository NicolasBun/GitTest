Detailed Description

template<int logBlockSize>
class embree::ObjectBinning< logBlockSize >

Single threaded object binner.

Finds the split with the best SAH heuristic by testing for each dimension multiple partitionings for regular spaced partition locations. A partitioning for a partition location is computed, by putting primitives whose centroid is on the left and right of the split location to different sets. The SAH is evaluated by computing the number of blocks occupied by the primitives in the partitions.
