Detailed Description

template<typename T>
struct embree::BVH2< T >::Node

BVH2 Node.

The nodes store for each dimension the lower and upper bounds of the left and right child in one SSE vector. This allows for a fast swap of the left and right bounds inside traversal for rays going from "right to left" in some dimension.
